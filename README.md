## 🔧 주요 기능
<br>

<details> 
  <summary><strong> 🗨️알림페이지: 모임 초대 알림, 모임 개설 완료 알림 등 유저에게 해당하는 알림이 보입니다.</strong></summary>
  <br/>
  <ul>
    <li>단어나 고민 상담 후 실시간으로 게시글, 댓글 외에도 실시간으로 빠른 소통이 가능한 창구입니다.</li>
    <li>여러 유저가 함께 대화할 수 있도록 메인 페이지에 다대다 채팅 기능을 구현했습니다.</li>
